#  프로젝트 생성

* [start.spring.io](start.spring.io)



* **Project**   
    * ``Maven`` : 라이브러리를 가져오고 빌드 관리 하는 툴
    * ``Gradle`` : 빌드배포도구(Build Tool) 

* **Spring Boot**
    * SNAPSHOT : 아직 만들고 있는 버전
    * M1 : 정식 아님

* **Project Metadata**
    * Group : 기업 명 적는거
    * Artifact : Build 결과물 적는거(프로젝트 명)
    
    
    
- **gradle** : 버전설정하고 라이브러리 가져 옴
