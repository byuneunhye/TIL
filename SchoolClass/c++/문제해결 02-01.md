# namespace에 대한 이해

- 이름공간(namespace)

```c
#include <iostream>

void func(void)
{
    std::cout<<"A회사가 정의 한 함수"<<std::endl;
}
void func(void)
{
    std::cout<<"B회사가 정의 한 함수"<<std::endl;
}
int main(void)
{
	func();
	return 0;
}
```
- 위의 코드를 실행하면 에러가 난다.   
- 매개변수의 반환형과 이름이 같기때문


```c
#include <iostream>

namespace A
{
	void func(void)
	{
		std::cout<<"A회사가 정의한 함수"<<std::endl;
	}
}

namespace B
{
	void func(void)
	{
		std::cout<<"B회사가 정의한 함수"<<std::endl;
	}
}

int main(void)
{
	A::func();
	B::func();
	return 0;
}
```
- 이건 에러가 안 난다.
- namespace별로 나눠져 실행되기 때문.
