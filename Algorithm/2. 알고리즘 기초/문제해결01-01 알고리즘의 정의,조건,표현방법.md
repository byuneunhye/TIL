# 알고리즘

**1. 정의** 

주어진 문제에서 필요한 답을 얻기위한 절차나 방법이다.

<br>

**2. 조건**  

 - 유한성 : 끝이 있어야 함       
 - 명확성 : 각 단계가 명확하게 명시 되야 함     
 - 입력   : 입력 없어도 됨   
 - 출력   : 하나 또는 그 이상의 출력 필요   
 - 효과성 : 각 단계들 효과적이어야 함.   

**3. 표현 방법**  

 - 자연어              :  우리 언어로 표현   
 - 의사코드            :  프로그래밍 언어와 비슷, 실제 코드가 아니므로 컴퓨터에서 실행 X    
 - 순서도              :  ``도형``과 순서를 알려주는 ``화살표`` 이용하여 명령 실행 순서 표현   
 - N-S 다이어그램      :  모듈화된 알고리즘 표현 효과적, 가독성 좋음
