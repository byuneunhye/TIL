# 표 만들 때 사용하는 태그

- 표는 \<table> 태그로 만든다.
- 행은  \<tr> 태그로 만든다.
- 열의 제목이 들어가는 셀은 \<th> 태그로 만든다. 
- 내용이 들어가는 셀은 \<td> 태그로 만든다.
- 각 열의 의미에 따라 \<thead>, \<tbody>, \<tfoot> 태그로 구분 지을 수도 있다.
- 가로로 이웃한 셀을 합칠 때에는 <b>\<colspan></b> 속성을 사용한다.
- 세로로 이웃한 셀을 합칠 때에는 <b>\<rowspan></b> 속성을 사용한다.
- 표 제목은 \<caption> 태그로 만든다. 

\* <b>행 : 가로</b>

\* <b>열 : 세로</b>
