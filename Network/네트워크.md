# 네트워크(Network)

- 의미 : 여러컴퓨터를 연결하여 상호 ``데이터를 주고받는`` 작업을 하는 ``컴퓨터들의 연결망``



- ``프로토콜``    
    - 네트워크 상에서 컴퓨터를 식별하는 방법   

    - 서로 다른 컴퓨터들이 ``대화``하기 위한 규칙과 절차



 - 네트워크 규모에 따른 분류
    - LAN : Local Area Network

    - MAN : Metropolitan Area Network

    - WAN : Wide Area Network

- 전송계층 단위 : 세그먼트 (Segment)
    - 네트워크 계층 단위 : 패킷 (Packet)

    - 데이터링크 계층 단위 : 프레임 (Frame)

        - 계층에 상관없이 전송단위를 ``패킷``이라 통칭

- 인터넷(Inter + Network)

    - 각 네트워크들 서로 연결
        - 3계층 장비인 ``라우터``로 연결

    - 서로 다른 운영체제와 통신 방식을 사용하는 각 네트워크가 서로 통신
        - 공통된 ``프로토콜``을 지원

    - 인터넷 통해 통신 위해
        - ``TCP/IP`` 통신 프로토콜 이용
- MAC (Machine Accesss Code)과 IP(Internet Protocol Address)
    - Lan 카드 또는 네트워크 인터페이스 카드
    - ``48비트 고유번호`` : 제조사 + 일련번호
    - ``2계층 주소``
